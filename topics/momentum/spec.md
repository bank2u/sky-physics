## แนวคิดที่จะสอน
โมเมนตัมรวมของระบบที่ไม่มีแรงภายนอกจะ "อนุรักษ์เสมอ" ไม่ว่าการชนจะเป็นแบบยืดหยุ่นหรือไม่ยืดหยุ่น
ขณะที่พลังงานจลน์รวมอนุรักษ์เฉพาะการชนแบบยืดหยุ่นเท่านั้น และในทุกการชน แรงที่วัตถุสองก้อนกระทำต่อกัน
มีขนาดเท่ากันเสมอ (กฎข้อ 3 ของนิวตัน) แม้มวลจะต่างกันมาก

## สิ่งที่อยากให้เด็กเห็น/เข้าใจ
- โมเมนตัมรวมก่อนชน = โมเมนตัมรวมหลังชน "เป็นตัวเลขเดียวกันเป๊ะ" ไม่ว่าปรับมวล/ความเร็ว/ชนิดการชนแบบใด
- พลังงานจลน์รวมลดลงหลังการชนแบบไม่ยืดหยุ่น (พลังงานหายไปเป็นความร้อน/เสียง/รูปทรง) แต่ยังคงเดิมในการชนแบบยืดหยุ่น
- แก้ misconception: รถใหญ่ชนรถเล็ก แรงกระทำต่อกัน "เท่ากัน" เสมอ (แรงดล J₁ = −J₂) ต่างกันแค่ผลของแรง (ความเร่ง a = F/m) เพราะมวลต่างกัน
- การชนแบบ slow-motion ปรับอัตราเวลาได้ ทำให้สังเกตช่วงกระทบซึ่งในของจริงเกิดเร็วเกินตาเห็น

## ค่าที่แสดง realtime
- โมเมนตัมรวมก่อนชน p(ก่อน) = m₁u₁ + m₂u₂ (kg·m/s)
- โมเมนตัมรวมหลังชน p(หลัง) = m₁v₁ + m₂v₂ (kg·m/s) — ตรงกับ p(ก่อน) เป๊ะทุกกรณี
- พลังงานจลน์รวมก่อนชน KE(ก่อน) (J)
- พลังงานจลน์รวมหลังชน KE(หลัง) (J) — น้อยกว่า KE(ก่อน) เมื่อชนไม่ยืดหยุ่น
- (บนตัวรถในซิม) มวล ความเร็วปัจจุบัน และโมเมนตัมของรถแต่ละคัน + ป้ายแรงดลเท่ากันตอนกระทบ

## สมการที่เกี่ยวข้อง
- อนุรักษ์โมเมนตัม: m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂
- สัมประสิทธิ์การคืนตัว: e = (v₂ − v₁) / (u₁ − u₂)
- พลังงานจลน์: KE = ½mv² (อนุรักษ์เฉพาะเมื่อ e = 1)
- กฎข้อ 3 (แรงดล): J₁ = m₁(v₁ − u₁) = −m₂(v₂ − u₂) = −J₂

## คำอธิบายแนวคิด (ขยายความ)
เมื่อไม่มีแรงภายนอกมากระทำต่อระบบ โมเมนตัมรวม (ผลรวมของ mv ของทุกวัตถุ) จะคงที่เสมอ นี่คือกฎการอนุรักษ์โมเมนตัม
ลองปรับมวลและความเร็วต้นของรถทั้งสองคัน แล้วสลับชนิดการชนดู จะเห็นว่าตัวเลข "โมเมนตัมรวมก่อนชน" กับ "โมเมนตัมรวมหลังชน"
เท่ากันเป๊ะทุกครั้ง ไม่ว่าจะชนแบบยืดหยุ่น (รถเด้งแยกจากกัน) หรือไม่ยืดหยุ่น (รถติดกันไปด้วยกัน) แต่พลังงานจลน์รวมนั้นต่างกัน:
การชนยืดหยุ่น (e=1) พลังงานจลน์รวมเท่าเดิม ส่วนการชนไม่ยืดหยุ่น (e<1) พลังงานจลน์บางส่วนเปลี่ยนไปเป็นความร้อน เสียง
และการเปลี่ยนรูปของวัตถุ ค่า KE(หลัง) จึงน้อยลง จุดสำคัญที่เด็กมักเข้าใจผิดคือคิดว่า "รถใหญ่ชนรถเล็ก แรงที่กระทำไม่เท่ากัน"
ความจริงตามกฎข้อ 3 ของนิวตัน ขณะกระทบ รถทั้งสองออกแรงต่อกันด้วยขนาดเท่ากันเสมอ (ลูกศรแรงดลสองอันในซิมยาวเท่ากันแม้มวลต่างกัน)
สิ่งที่ต่างคือ "ผล" ของแรง เพราะความเร่ง a = F/m รถเล็กมวลน้อยจึงเปลี่ยนความเร็วมากกว่า ทั้งที่รับแรงเท่ากัน
ปรับอัตราเวลาให้ช้าลงเพื่อดูช่วงกระทบซึ่งในการทดลองจริงเกิดเร็วเกินกว่าจะสังเกตได้
