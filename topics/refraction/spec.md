## แนวคิดที่จะสอน
กฎการหักเหของแสง (กฎของสเนลล์) ที่อธิบายความสัมพันธ์ระหว่างมุมตกกระทบ มุมหักเห และดัชนีหักเหของตัวกลางทั้งสอง รวมถึงปรากฏการณ์มุมวิกฤตและการสะท้อนกลับหมดภายใน (Total Internal Reflection) เมื่อแสงเดินทางจากตัวกลางหนาแน่นกว่าไปยังตัวกลางเบาบางกว่า

## สิ่งที่อยากให้เด็กเห็น/เข้าใจ
- ทิศทางการเบนของรังสีหักเหเมื่อแสงเข้าตัวกลางที่มีดัชนีหักเหมากกว่า (เบนเข้าหาเส้นแนวฉาก) หรือน้อยกว่า (เบนออกจากเส้นแนวฉาก)
- เมื่อมุมตกกระทบเพิ่มขึ้นเรื่อยๆ (จากตัวกลางหนาแน่นไปเบาบาง) มุมหักเหจะเพิ่มเร็วกว่าจนถึงมุมวิกฤต แล้วรังสีหักเหจะ "หายไป" กลายเป็นสะท้อนกลับหมดภายใน
- มุมวิกฤตขึ้นกับอัตราส่วนดัชนีหักเหของตัวกลางทั้งสอง

## ตัวแปรที่ปรับได้
| ตัวแปร | ช่วงค่า | หน่วย | ค่าเริ่มต้น |
|--------|---------|-------|-------------|
| มุมตกกระทบ (θ₁) | 0–90 | องศา | 30 |
| ตัวกลางต้นทาง | อากาศ (n=1.00) / น้ำ (n=1.33) / แก้ว (n=1.50) / เพชร (n=2.42) | - | แก้ว |
| ตัวกลางปลายทาง | อากาศ (n=1.00) / น้ำ (n=1.33) / แก้ว (n=1.50) / เพชร (n=2.42) | - | อากาศ |

## ค่าที่แสดง realtime
- มุมหักเห θ₂ (องศา)
- มุมวิกฤต θc (องศา, แสดงเมื่อ n₁ > n₂)
- สถานะ "สะท้อนกลับหมดภายใน" เมื่อ θ₁ > θc

## สมการที่เกี่ยวข้อง
- n₁ sinθ₁ = n₂ sinθ₂ (กฎของสเนลล์)
- sinθc = n₂ / n₁ (เมื่อ n₁ > n₂)

## คำอธิบายแนวคิด (ขยายความ)
เด็กจำสูตรสเนลล์ได้แต่ทายทิศทางการเบนผิดอยู่เสมอ และมุมวิกฤตกับการสะท้อนกลับหมดเป็นเรื่องที่วาดมือแล้วเด็กไม่ค่อยเชื่อจนกว่าจะได้เห็นเอง simulation นี้ให้ลากปรับมุมตกกระทบต่อเนื่อง เห็นรังสีหักเหเบนไปเรื่อยๆ ตามกฎของสเนลล์ จนถึงมุมวิกฤตพอดี รังสีหักเหจะ "หายไป" และกลายเป็นรังสีสะท้อนเต็มพลังงานแทน — โมเมนต์ที่รังสีหักเหหายวับไปนี้คือจุดร้องอ๋อของเรื่องนี้ เลือกคู่ตัวกลางจากวัสดุจริง (อากาศ/น้ำ/แก้ว/เพชร) ทำให้เห็นว่าเพชรมีมุมวิกฤตแคบมาก (~24°) ซึ่งเป็นเหตุผลที่เพชรเจียระไนแล้ววาววับ
