คำอธิบายแนวคิด

หลักการซ้อนทับ (superposition) กล่าวว่าเมื่อคลื่นสองขบวนเคลื่อนที่ผ่านกัน แต่ละจุดบนตัวกลางจะมีการกระจัดเท่ากับผลรวมทางพีชคณิตของการกระจัดของคลื่นแต่ละขบวน ณ จุดนั้น (y = y₁ + y₂) โดยคลื่นแต่ละขบวนจะเดินทางต่อไปตามเดิมหลังผ่านกัน ไม่ได้ "ชนกัน" แต่อย่างใด — แผงบนของ simulation แสดงคลื่นสองขบวนที่มีความถี่และแอมพลิจูดเท่ากันเคลื่อนที่สวนทางกัน สังเกตเส้นผลรวม (เส้นทึบหนา) จะแบนราบเมื่อยอดคลื่นซ้อนกับท้องคลื่นพอดี (หักล้าง) และสูงเป็นสองเท่าเมื่อยอดซ้อนยอด (เสริม) สลับกันไปตามเวลา

เมื่อคลื่นอาพันธ์ (coherent) จากสองแหล่งกำเนิดที่ห่างกันเป็นระยะ d แผ่ออกไปพร้อมกัน (เช่น การทดลองช่องคู่ของ Young หรือถังคลื่น) จุดใดบนจอที่ผลต่างระยะทางจากสองแหล่ง (Δ = d sinθ) เท่ากับจำนวนเต็มเท่าของความยาวคลื่น (Δ = nλ) จะเกิดการแทรกสอดเสริม (ปฏิบัพ, สว่าง) ส่วนจุดที่ Δ เท่ากับจำนวนเต็มบวกครึ่งของความยาวคลื่น (Δ = (n+½)λ) จะเกิดการแทรกสอดหักล้าง (บัพ, มืด) แผงล่างแสดงคลื่นวงกลมจากสองแหล่งกำเนิดและลวดลายแถบมืด-สว่างที่เกิดขึ้นจริงบนจอ ปรับความถี่หรือระยะห่างช่องคู่แล้วจะเห็นแถบเลื่อนและบีบ/ขยายทันที เพราะระยะห่างระหว่างแถบ (Δy) แปรผันตรงกับ λ และแปรผกผันกับ d
