## แนวคิดที่จะสอน
ผลักกล่องบนพื้นแล้วปล่อย สังเกตว่าอะไรทำให้กล่องช้าลงจนหยุด — ไม่ใช่ "แรงหมด" แต่เป็น
**แรงเสียดทาน** ที่ต้านการเคลื่อนที่อยู่ตลอด เมื่อค่อยๆ ลดสัมประสิทธิ์แรงเสียดทาน μ ลงจนเป็น 0
จะเห็นว่ากล่องที่ถูกผลักครั้งเดียวแล้วปล่อย จะไถลต่อไปเรื่อยๆ ไม่มีวันหยุดเอง โดยไม่ต้องออกแรงอีกเลย
นี่คือหัวใจของกฎข้อ 1 ของนิวตัน (ความเฉื่อย) ที่ทดลองจริงในห้องไม่ได้เพราะโลกจริงมีแรงเสียดทานเสมอ

## สิ่งที่อยากให้เด็กเห็น/เข้าใจ
- ความเชื่อแบบอริสโตเติล "มีแรงถึงจะเคลื่อนที่ หมดแรงก็หยุด" ผิด — สิ่งที่ทำให้หยุดคือแรงเสียดทาน ไม่ใช่การหมดแรง
- **จุด "อ๋อ":** ลาก μ ลงมาที่ 0 (หรือกดพื้นแบบ "ลื่นไร้แรงเสียดทาน") → กดผลักกล่องแวบเดียวแล้วปล่อย →
  กล่องไถลต่อด้วยความเร็วคงที่ตลอดไป แรงลัพธ์ = 0 แต่ยังเคลื่อนที่ (ไม่ต้องมีแรงพยุงให้เคลื่อนที่)
- เมื่อ μ > 0 และปล่อยมือ แรงลัพธ์ชี้สวนทางการเคลื่อนที่ → ความเร่งติดลบ → ช้าลงจนหยุด
- ขณะที่ผลัก กล่องกำลังเคลื่อนที่ แรงลัพธ์ = แรงผลัก − แรงเสียดทาน และความเร่ง a = แรงลัพธ์ / มวล
- ถ้าแรงผลักน้อยกว่าแรงเสียดทานสูงสุด กล่องหนักบนพื้นหยาบก็ผลักไม่ไป (แรงเสียดทานสถิตต้านไว้พอดี)

## ตัวแปรที่ปรับได้
| ตัวแปร | ช่วงค่า | หน่วย | ค่าเริ่มต้น |
|--------|---------|-------|-------------|
| แรงผลัก (ขณะกดปุ่มผลัก) | 0–50 | N | 25 |
| สัมประสิทธิ์แรงเสียดทาน μ | 0–0.6 | (ไม่มีหน่วย) | 0.30 |
| มวลของกล่อง | 1–8 | kg | 2 |

หมายเหตุการตัดสินใจ (ให้ครูรีวิว):
- ช่วง μ 0–0.6 ครอบคลุมค่าจริง: น้ำแข็ง ≈0.05, ไม้ขัดมัน ≈0.2, พื้นปูน ≈0.4, ยางบนถนน ≈0.6 และลงถึง 0 เพื่อกรณีอุดมคติไร้แรงเสียดทาน
- มีปุ่มเลือกชนิดพื้น (preset) ที่ตั้งค่า μ ให้อัตโนมัติ เพื่อให้ครูสลับสถานการณ์เร็ว รวมถึงปุ่ม "ลื่นไร้แรงเสียดทาน" (μ=0)
- ใช้ g = 10 m/s² ตามธรรมเนียม ม.ต้น (แรงเสียดทานสูงสุด = μ·m·g); ที่ μ, m สูงสุด แรงเสียดทาน = 48 N < 50 N จึงยังผลักให้เคลื่อนที่ได้เสมอ
- ใช้ μ เดียวแทนทั้งแรงเสียดทานสถิตและจลน์ เพื่อความเรียบง่ายระดับ ม.ต้น
- แรงผลักออกด้วยการ "กดค้าง" ที่ปุ่มผลัก (กดแวบเดียวแล้วปล่อยได้) เพื่อให้เกิดจังหวะ "ผลักแล้วปล่อย" ตรงกับจุดอ๋อ

## ค่าที่แสดง realtime
- แรงลัพธ์ (N) — ขนาดและทิศ (บวก = ทิศผลัก)
- ความเร่ง a (m/s²)
- แรงเสียดทาน (N) — ต้านการเคลื่อนที่เสมอ
- ความเร็ว v (m/s)
- สถานะ: กำลังเคลื่อนที่ / หยุดนิ่ง

## สมการที่เกี่ยวข้อง
- แรงลัพธ์: F<sub>รวม</sub> = F<sub>ผลัก</sub> − f (ขณะเคลื่อนที่)
- กฎข้อ 2 ของนิวตัน: a = F<sub>รวม</sub> / m
- แรงเสียดทาน: f = μ · m · g

## คำอธิบายแนวคิด (ขยายความ)
เด็กเกือบทุกคนคิดแบบอริสโตเติลว่า "ต้องออกแรงตลอดของถึงจะเคลื่อนที่ พอหมดแรงมันก็หยุดเอง" เพราะในโลกจริง
เข็นรถ ผลักกล่อง พอหยุดออกแรงมันก็ช้าลงแล้วหยุดทุกที แต่สิ่งที่ทำให้มันหยุดไม่ใช่ "แรงหมด" —
เป็นแรงเสียดทานจากพื้นที่คอยต้านการเคลื่อนที่อยู่ตลอดเวลาต่างหาก ในภาพจำลอง ลูกศรสีชมพูคือแรงเสียดทาน
ที่ชี้สวนทางการเคลื่อนที่เสมอ พอปล่อยมือ (ไม่มีแรงผลักแล้ว) แรงลัพธ์จึงเหลือแต่แรงเสียดทานที่ฉุดให้ช้าลงจนหยุด

ทีนี้ลองลากแถบ μ ลงมาที่ 0 หรือกดปุ่มพื้น "ลื่นไร้แรงเสียดทาน" แล้วกดผลักกล่องแวบเดียวแล้วปล่อย —
กล่องจะไถลต่อไปด้วยความเร็วคงที่ ไม่ช้าลง ไม่หยุด ตลอดไป ทั้งที่ไม่มีแรงอะไรผลักมันเลย (แรงลัพธ์ = 0)
นี่คือกฎข้อ 1 ของนิวตัน: วัตถุที่กำลังเคลื่อนที่จะเคลื่อนที่ต่อไปด้วยความเร็วคงที่ ถ้าไม่มีแรงลัพธ์มากระทำ
การเคลื่อนที่ไม่ต้องใช้แรงมาพยุง — แรงมีไว้เปลี่ยนการเคลื่อนที่ (ทำให้เร็วขึ้น ช้าลง หรือเลี้ยว) เท่านั้น
การทดลองนี้ทำจริงในห้องเรียนไม่ได้ เพราะเราสร้างพื้นที่ไร้แรงเสียดทานสมบูรณ์ไม่ได้ simulation จึงช่วยให้
"เห็น" สิ่งที่ธรรมชาติซ่อนไว้เสมอได้
